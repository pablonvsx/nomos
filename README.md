# Nomos

## Academic Context

Nomos is a technological product developed as part of a Master's research project within the **Postgraduate Program in Geography (PPGEO)** at the **Federal University of Pernambuco (UFPE)**.

* **Author:** Pablo Guilherme de Melo Neves
* **Advisor:** Professor Lucas Costa de Souza Cavalcanti, Ph.D.
* **Research Group:** PAISAGEO - Geografia de Paisagens Tropicais

## About the Project

**Nomos** is a mobile application designed to assist researchers in **Landscape Mapping**. It acts as a methodological mediation tool, replacing traditional analog records with a structured and standardized digital data collection process.

The app allows for the systematic recording of geoecological variables and provides automatic taxonomic classification of landscape units based on predefined geosystemic rules.

## Tech Stack

* **Core Framework:** React Native with Expo
* **Programming Language:** TypeScript
* **Local Database:** SQLite
* **UI Engine:** React Native Paper
